## Configuration


```bash
# Cloner le dépot
git clone https://github.com/pierretl/jeux.git

# Installation des dépendances
npm install

# Pendant le dev : (le navigateur s'ouvre automatiquement sur http://localhost:3000)

npm watch

# Pour la production
# Cette commande rebuild tout, compresse le css 
npm build
```
